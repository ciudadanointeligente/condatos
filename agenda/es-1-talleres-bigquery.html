Google BigQuery para Datos Abiertos

Google


Octubre 1 de 1600 - 1900


Su objetivo es presentar dos proyectos de uso de datos abiertos desarrollados por Google. El primero es BigQuery, un servicio que permite el análisis masivo de grandes volúmenes de datos utilizando el poder de procesamiento de la infraestructura de Google. El segundo es The GDELT Project, una base de datos en tiempo real de la sociedad global para investigación abierta. Al final, se realizará un taller enfocándose en mostrar a los participantes cómo implementar proyectos de esta magnitud utilizando herramientas para escalar sus análisis de bases de datos y para poder trabajar con gobiernos a medida que van abriendo sus bases de datos.
